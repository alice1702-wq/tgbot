# Страховой Гид — бот для зумеров

> Telegram-бот, который объясняет основы страхования на живых примерах, мемах и брейнрот-персонажах.

# Работа с ботом

- Для начала работы вбейте никнейм бота @Zoomer_Insurance_bot в поисковой строке и нажмите кнопку старт

## Цель проекта

Помочь подросткам и молодым людям (11–18 лет) разобраться в страховании через интерактивные кейсы, тесты и простой язык. Бот показывает, какие риски существуют в повседневной жизни, как работает страхование и в каких ситуациях оно реально спасает.

## Функциональность

- **Глоссарий** — термины с картинками и пояснениями по‑зумерски (риск, страховой случай, выплата, полис, страховая сумма, срок, исключения).
- **Тест по терминам** — 6 вопросов для проверки знаний. После каждого ответа — пояснение.
- **Кейсы** — 5 реальных ситуаций с участием известных персонажей (Тунг‑тунг Сахур, Дрысясиси, Хаги‑Ваги, Ам-Ням, Амогус). Каждый кейс содержит:
  - Картинку
  - Голосовое сообщение с описанием
  - Три варианта ответа
  - Реакцию «Правильно» / «Неправильно» + отдельное сообщение с матчастью
- **Босс‑тест** — 5 вопросов по всем темам. При 4–5 правильных ответах пользователь получает бонус (PDF‑файл).

## 🛠 Технологии

- Python 3.12+
- aiogram 3.x
- Base64 (PDF как часть кода)
- Голосовые сообщения и картинки через Telegram API
